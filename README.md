# Email Extractor Automation Script

## Description
This Python script extracts all email addresses from a text file and saves them into another file.

## Features
- Reads text from input file
- Extracts emails using regex
- Removes duplicate emails
- Saves output to another file

## Technologies Used
- Python 3
- Regular Expressions (re)
- File Handling

## How to Run
```bash
python email_extractor.py
```

## Input File
- input.txt (contains text with emails)

## Output File
- emails.txt (contains extracted emails)
